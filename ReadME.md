# Cursed Colors 

### Humans can't reliably recall colors. So, this is a browser based online game made in vanilla js to see how good (or bad) you are at remembering those colors.
---
# How to Run!
- **Locally:**
- Clone the Repo
```
git clone https://github.com/adicoding-js/cursedcolors.git
```
- and open index.html directly on your browser and play there.

- **Online:**
- Just go to -> [Cursed Colors](https://cursedcolors.netlify.app/) and play directly here

---
# How to play!
- 1) Pick a difficulty — **Easy** (5 seconds) or **Hard** (2 seconds)
- 2) CLick Start. The card fills with a random color, memorize it
- 3) When the timer hits 0, use the two color strips on the left to     recreate it
- 4) Submit your guess and see how close you got
- 5) Do it 4 more times, then get your session score

---
# Scoring System
- Each round is scored by how close your guess is to the target in RGB space (euclidean distance). The closer you get, the higher the score.
 
| score | label |
|-------|-------|
| 90–100% | Dialed In |
| 70–89% | Getting There |
| below 70% | Cursed |
